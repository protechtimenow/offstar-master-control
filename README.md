## 💻 **Local Development**

### **Quick Start:**
```bash
# Clone the repository
git clone https://github.com/protechtimenow/offstar-master-control.git
cd offstar-master-control

# Install dependencies
npm install

# Start the control hub
npm start
```

